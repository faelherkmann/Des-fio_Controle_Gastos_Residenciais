# Household Expenses Web

Front-end em React + TypeScript para o desafio de controle de gastos residenciais.

## Como executar

1. Instale as dependências com `npm install`.
2. Inicie o front-end com `npm run dev`.
3. Se necessário, ajuste `VITE_API_BASE_URL` para apontar para o backend ASP.NET Core.

A API padrão esperada é `http://localhost:5014`.
